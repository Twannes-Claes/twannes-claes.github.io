Pepper Robot

C++
Python
MQTT
VUE
For my summer job, I got hired by a teacher at Howest University to help them in re-programming a Pepper Robot that would be used in a hotel.

The robot needed functionality to work with the website displayed on its tablet. I programmed the functionality to make her move, speak and display the site, allow editing of settings, and integrated face recognition and animations.

I initially developed the robot control system using C++, where I successfully implemented command execution through a library that interfaces directly with the robot. The original plan was to connect this system to a Vue.js based website developed by my supervisor, allowing the site to send commands to the robot. However, due to integration issues between the website and the C++ backend, I transitioned the rest of the project to Python for improved connectivity and ease of use. Throughout the project, I also contributed to the Vue.js frontend by adding content and supporting features needed to control the robot remotely.
